2024/12

  朋友，你好！

  这里是 SWL，北京大学信科学院23届本科生。

  本站已经持续运营一年了，值此佳际，鄙人 SWL 特将许久未更新的 vivia 主题换为 a4 主题。正所谓新衣辞旧岁，本来冷冷清清的小站焕然一新，就像蓬头垢面的男友突然变得衣冠楚楚，不禁让人夸赞他的成长，尽管这些装饰都出自我手。这也在侧面不得不证明了外在感受的重要性，“酒香不怕巷子深”的含金量还在下降。

  不过鄙人也不是什么好种，鄙站也酿不出什么好酒。尽管如此，这一年的成长依然有目共睹，这篇博客就是一条线索，浏览它的你和创办它的我都是见证者。很庆幸，我能够克服我的秘密主义，向互联网投入一篇又一篇胡言乱语，这些文字已经成了牵挂我心之锚，在风雨飘摇的精神之海上将我牢牢拴住，不至于流浪。

  我很喜欢新的主题。vivia 主题是在去年12月投入使用的，是本博客沿用了一年的主题。当时本博客的取向还是技术类博客，从这一角度看，vivia 还是蛮合适的。当时的我还天真地认为我能够坚持不懈地维护下去，实际上，直到今年9月，本博客除了一篇冷饭，完全没有后文。今年9月是一个很重要的节点，因为我终于想起来了我还有一个域名和与之关联的 github 仓库。自此开始，本博客的主旨就变为了 SWL 的思想跑马场。尽管不是小马，就是病马，但终究是有意义的马。

  总之无论如何，祝你阅读愉快！

  但是！如果你认为这是一个技术向博客，那你就错了。就目前而言，这个博客仅仅是对我日常随笔的记录。我是不思考就活不下去的怪人，是什么都敢想什么都敢说的诗人。

  目前的计划是尽量多尝试，多更新。多写读后感，多独立创作，尽可能把自己的思想好好落在文字间。

  如果你爱，那固然好；如果你不爱，你当然可以把这个博客放进收藏夹里，说不定你想要的内容就长出来了——又有什么是不会改变的呢？

  虽然这么说，大概不会有人赛博迷路到这里吧？（笑）

  总之无论如何，祝你阅读愉快！
